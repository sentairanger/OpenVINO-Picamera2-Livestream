About OpenVINO Picamera2 Livestream UI

Web Interface for the Raspberry Pi Camera Module and the OpenVINO Toolkit

This project uses the Picamera2 library to take images and the OpenVINO toolkit classifies them, detects text or recognizes vehicles using pretrained models. This project will require the Pi Camera module and the Intel NCS2 (Neural Compute Stick 2)

GitHub Repository

Capture Images

Capture images using a single button and save them to a gallery.

Classify Images

After capturing images, use OpenVINO's classification model to classify the images

Detect Text

You can also use OpenVINO's text detection model to detect text in an image.

Recognize Vehicles

You can also use OpenVINO's vehicle detection and recognition models to detect and recognize vehicles.

Gallery

The gallery in this project will allow you to view the text detected in the image or the vehicle that was detected in the image.
